# discordbot-BSPlaylistManager

## 説明
discord上でBeat Saberのプレイリストを管理するbotです。  
herokuとdropboxを利用します。


## コマンドリスト
### /add
```
/add [URL]
```
URLは、[bsaber.com](https://bsaber.com/)の曲ページのURLです。  
指定の曲がリストに追加されます。

### /delete
```
/delete [URL]
```
URLは、[bsaber.com](https://bsaber.com/)の曲ページのURLです。  
指定の曲がリストから削除されます。

### /dl (or /download)
```
/dl
```
プレイリストのリンクを返します。

### /ping
```
/ping
```
pongと返します。
